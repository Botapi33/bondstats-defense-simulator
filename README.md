# BondStats – Defense Bond Simulator

A structured simulation tool to understand how defense spending impacts bond markets.

## Overview

This tool models the interaction between:
- Fiscal expansion
- Bond issuance
- Yield dynamics
- Market expectations

## Purpose

Not to predict markets, but to structure thinking.

## Features

- Scenario-based simulation
- Yield direction modeling
- Yield curve impact
- Supply pressure analysis

## Philosophy

Markets are driven by expectations, not just data.
